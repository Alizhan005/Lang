# Инструкции по копированию страниц из Figma

## Как скопировать страницы 3-8 из Figma:

### Вариант 1: Через Figma Desktop
1. Откройте Figma Desktop приложение
2. Откройте файл LangBridge
3. В левой панели выберите страницу 3 (или 4, 5, 6, 7, 8)
4. Сообщите мне "готово" или "скопируй страницу 3"
5. Я получу дизайн и заполню соответствующий компонент

### Вариант 2: Через ссылки
Отправьте ссылки на страницы 3-8 в формате:
```
https://www.figma.com/design/QKTMfb1I2xW1FsNgZ8Tk4v/LangBridge?node-id=...
```

### Структура проекта:
- `src/pages/Page3.jsx` - Страница 3
- `src/pages/Page4.jsx` - Страница 4
- `src/pages/Page5.jsx` - Страница 5
- `src/pages/Page6.jsx` - Страница 6
- `src/pages/Page7.jsx` - Страница 7
- `src/pages/Page8.jsx` - Страница 8

Все компоненты готовы к заполнению дизайном из Figma.
