# LangBridge - AI-Powered Kazakh Language Learning Platform

React приложение для изучения казахского языка с использованием AI.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу http://localhost:3000

## Структура проекта

```
src/
  ├── pages/          # Страницы приложения
  │   ├── Home.jsx    # Главная страница
  │   ├── Signup.jsx  # Страница регистрации
  │   ├── Page3.jsx   # Страница 3
  │   ├── Page4.jsx   # Страница 4
  │   ├── Page5.jsx   # Страница 5
  │   ├── Page6.jsx   # Страница 6
  │   ├── Page7.jsx   # Страница 7
  │   └── Page8.jsx   # Страница 8
  ├── styles/         # Стили для каждой страницы
  └── App.jsx         # Главный компонент с роутингом
```

## Технологии

- React 18
- React Router DOM
